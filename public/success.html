How to Fix the Missing Transaction Reference

The error occurs because the transaction reference (tx_ref) is not being passed correctly in the URL when redirecting to the success page.

Make sure your server redirects to: http://yoursite.com/payment-success?tx_ref=UNIQUE_REFERENCE
